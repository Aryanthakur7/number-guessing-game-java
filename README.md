# 🎯 Number Guessing Game (Java)

A console-based number guessing game built using Java.  
The program generates a random number and challenges the user to guess it within limited attempts.

---

## 🚀 Features

- Random number generation
- User input handling
- Attempt counter
- Hint system (Too High / Too Low)
- Clean console interaction
- Basic OOP structure

---

## 🛠️ Tech Used

- Java
- IntelliJ IDEA
- Core Java Concepts
- Loops & Conditional Logic

---

## 🧠 Concepts Practiced

- Object-Oriented Programming
- Control Flow
- Random Class
- Input Handling (Scanner)
- Game Logic Implementation

---

## ▶️ How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the `Main.java` file
4. Start guessing!
